# RPG Torch Puzzle

Um puzzle interativo inspirado em RPGs medievais.

## 🎮 Objetivo
Apagar todas as tochas do círculo.

Ao clicar em uma tocha:
- ela muda de estado,
- e as tochas adjacentes também.

## 🔥 Tecnologias
- HTML5
- CSS3
- JavaScript

## 🎵 Recursos
- Áudio ambiente
- Sons interativos
- Sistema de vitória
- Atmosfera medieval
